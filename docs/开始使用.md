---
title: 开始使用
---

此教程将教你安装、配置及使用SRA。

## 1. 下载、安装

前往 [下载页面](../pages/download) 选择合适的节点、版本下载。

::: warning 警告
虽然SRA CE理论上不需要运行环境，但实际上仍需要[Microsoft Edge WebView2](https://developer.microsoft.com/zh-cn/microsoft-edge/webview2)。

如果发现无法运行，可能为缺失该环境导致。

本程序理论上可在Windows 7/8/8.1运行，但 Python 后端可运行性未知。
:::

### 安装包
- 下载完成后，在浏览器的下载窗口或文件资源管理器中打开。按照指示完成安装。

- 安装完成后，在开始菜单搜索 StarRailAssistant 即可看到程序。

## 2. 添加信任列表
[前往SRA官网介绍界面](https://starrailassistant.top/getstarted/getstarted.html#%E6%B7%BB%E5%8A%A0%E4%BF%A1%E4%BB%BB) 查看如何添加信任。

## 3. 界面

![SRA-CE主界面](/SRA-Home-C.png)

- 左侧：导航栏
- 右侧：主内容

当前展示的是主页的内容，点击其他选项即可切换其他页面。

- 头部：展示后端（SRA-cli）的运行状态、运行任务以及重启后端（SRA-cli）。
- 快速访问：提供选项供用户快速访问
- 配置管理：可以新建配置、选择配置、删除配置。

## 4. 任务配置
![SRA-CE任务配置](/SRA-Tasks-C.png)

此步骤与官方SRA的配置步骤无异。你可以通过上方按钮来保存配置、创建新配置，以及执行配置。

![SRA-CE清体力配置](/SRA-Tasks-C-2.png)

与官方不同的是，在 `任务列表` 中，你可以将鼠标移动到具体任务上，进行上下移动的操作。

### 旷宇纷争

该配置项与原SRA无异。清空其他任务，在游戏中打开该界面，选择开始执行。

货币战争同样支持接管模式。要以接管模式执行货币战争任务，您必须处于货币战争的准备阶段界面，确保商店界面是关闭的。

## 5.执行任务

![SRA-CE配置管理](/SRA-TaskManager-C.png)

你可以在这里选择要执行任务的顺序，然后点击上方的 `开始任务` 开始执行。

在运行过程中， `SRA` 会主动确保 `崩坏：星穹铁道` 窗口置于激活状态。除非终止任务，否则不建议切换窗口。

::: warning 注意
*SRA* 官方版本 (低于v2.0版本) 支持 `F9` 快速结束任务，但目前2.0的 *SRA* 以及截止到目前的 *SRA-CE* 仍未支持快捷键。
:::
