# Getting Started

Welcome to StarRailAssistant Community Edition!

## Introduction

StarRailAssistant Community Edition is a modern rebuild of the original StarRailAssistant, using Tauri + Vue 3 technology stack to provide a faster, lighter, and more modern user experience.

## Features

- **Lightweight & Efficient**: Built with Rust + Tauri, frontend size only ~10MB, memory usage as low as 50MB
- **Real-time Feedback**: Complete logging system and status monitoring
- **Fully Compatible**: Retains all core SRA features
